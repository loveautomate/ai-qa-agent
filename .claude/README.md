# Default **playwright-cli** skills location

[`playwright-cli install --skills`](https://playwright.dev/docs/getting-started-cli) writes here: `.claude/skills/playwright-cli/` (this repo tracks it so clones get skills without re-running install).

Refresh after upgrading `@playwright/cli`: `npm run playwright-cli:skills`

Repository layout (how this folder fits with tests, chatmodes, MCP): [README.md — Architecture](../README.md#architecture).
